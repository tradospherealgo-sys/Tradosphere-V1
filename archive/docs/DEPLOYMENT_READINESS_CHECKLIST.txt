================================================================================
ROLLBACK PROCEDURE
================================================================================

If anything goes wrong:

Option 1: Quick Rollback (Database)
  1. Stop the server:
     pkill -f "tradosphere_saas_server"
  
  2. Restore database:
     rm tradosphere.db
     cp tradosphere.db.backup.latest tradosphere.db
  
  3. Restart server with old code:
     git revert HEAD
     python3 tradosphere_saas_server.py

Option 2: Git Rollback
  1. Revert the commit:
     git revert HEAD --no-edit
     git push origin main
  
  2. Railway will auto-deploy the reverted version

Option 3: Database Restore Only
  1. Keep the code, revert database:
     cp tradosphere.db.backup.latest tradosphere.db
  
  2. Clear error state:
     pkill -f "tradosphere_saas_server"
     python3 tradosphere_saas_server.py
